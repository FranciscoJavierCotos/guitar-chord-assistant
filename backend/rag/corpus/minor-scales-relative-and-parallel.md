## The natural minor scale and relative keys

Every major key has a relative minor: the minor key that shares its exact key
signature (same notes, no added sharps/flats) but starts on the major scale's 6th
degree instead of its 1st. A major's relative minor is F# minor; C major's is A
minor. Because the notes are identical, a chord progression written "in A minor"
and one written "in C major" can use the very same chords — the difference is which
chord feels like home (the tonic). This is why a guitarist can often turn a
cheerful-sounding progression "sadder" just by starting and ending on the relative
minor chord instead of the major I, without changing a single note in between.

## Parallel minor and harmonic/melodic variants

A parallel minor shares the same tonic as a major key but a different key signature
— C major and C minor both start on C, but C minor flattens the 3rd, 6th, and 7th
degrees relative to C major. Borrowing chords from the parallel minor (e.g. using a
bVI or bVII chord in an otherwise major-key song) is a common way to add color
without leaving the home key entirely. Natural minor is also frequently altered:
harmonic minor raises the 7th degree to create a stronger pull back to the tonic
(at the cost of an unusual augmented 2nd gap), and melodic minor traditionally
raises both the 6th and 7th degrees ascending. These variants exist because the
natural minor's lowered 7th doesn't produce a strong leading tone, so composers
adjusted the scale when they wanted that pull.
